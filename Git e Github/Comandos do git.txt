======= Comandos para baixar repositorio no pc=======
1. pegar o link do repositorio no github;
2. abrir o git bash na pasta desejada
3. utilizar comandos: git clone (cole o link aqui sem os parenteses);

======= Comandos para comitar o repositorio que foi feito alteração=======

1. git add . (ou git add -a) = adicionar todos os arquivos
2. git commit -m "coloque um comentario aqui" = comitar os arquivos novos
3. git push origin main = publicar os commits para a nuvem
